# MICompete

Find the perfect teammate for B-school case competitions. Create your profile, search for ongoing competitions on Unstop and Mettl, and chat with potential partners in real-time.

## Features

- **Profile Creation**: Set up your professional/student profile with details that matter.
- **Competitions**: View ongoing matches.
- **Teammate Finder**: Discover and connect with students whose skills match what you need.
- **Real-Time Chat**: Message directly within the platform.
- **Built with**: React, Vite, Tailwind CSS, Firebase.

## Setup

1. Configure your environment variables in `.env`.
2. Run `npm install`
3. Run `npm run dev`
